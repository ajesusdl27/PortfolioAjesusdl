## Resumen

Un portfolio de desarrollador hecho a mano que destaca:

- **Cuadrícula bento** — Sistema asimétrico y responsive de tarjetas para reforzar la jerarquía visual
- **Diseño glassmorphism** — Tarjetas translúcidas con alto blur de fondo y bordes con degradado
- **Animaciones al hacer scroll** — Entradas escalonadas con fade-in-up usando Framer Motion + Intersection Observer
- **Arquitectura de islas** — Componentes React hidratados solo cuando son visibles para optimizar el rendimiento
- **Colecciones de contenido** — Proyectos gestionados como archivos Markdown locales con esquemas tipados
