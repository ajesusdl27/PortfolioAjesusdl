## Overview

A handcrafted developer portfolio showcasing:

- **Bento grid layout** — Asymmetric, responsive card system for visual hierarchy
- **Glassmorphism design** — Translucent cards with high backdrop blur and gradient borders
- **Scroll animations** — Staggered fade-in-up reveals using Framer Motion + Intersection Observer
- **Island architecture** — React components hydrated only when visible for optimal performance
- **Content Collections** — Projects managed as local Markdown files with typed schemas
